1. Checkout the repository
```sh
git clone git@github.com:lnis-uofu/ECE_5710_6710_F23.git
cd 5710_6710_F23
```
2. Setup the environment variables in a `csh`/`tcsh` shell
```csh
source setup_enc.csh
```
